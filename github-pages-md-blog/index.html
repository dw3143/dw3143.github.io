欢迎 👋

这是一个用 GitHub Pages 搭建的单文件极简网站，支持直接读取仓库里的 .md 文章。

把你的 Markdown 放在 posts/ 目录，例如 posts/hello-world.md。

在 posts/posts.json 中登记文章的 slug、标题和日期，前端会自动渲染列表与详情。
